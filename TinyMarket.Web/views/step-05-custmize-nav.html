You can specify more buttons in the navigation in your step's controller.

Below we have 3 buttons

  • The first one will navigate to the welcome step (first step).
  • The second one will navigate to the summary step (last step).
  • The thrid one is dynamical, it will navigate to the step you selected.

Also note that the controller of this step was defined in a separeated file with the controller name specified.

Select one step and click "Go Dynamic" button in navigation will lead you to that step.
